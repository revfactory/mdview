# Screenshots

이 디렉토리에는 README 와 문서에서 사용되는 스크린샷·GIF·이미지를 보관합니다.

## 파일 명명 규칙

- `hero.png` — README 상단 메인 스크린샷 (권장 비율 16:9, 1600x900)
- `feature-{기능명}.png` — 기능별 스크린샷 (예: `feature-hwp-import.png`)
- `demo-{시나리오}.gif` — 동작 GIF (10초 이내, 1200px 이하 폭 권장)

## 최적화

업로드 전 다음을 고려해 주세요:

- PNG: [TinyPNG](https://tinypng.com) 또는 `pngquant` 로 압축
- GIF: 가능하면 `mp4`/`webm` 으로 변환 후 README 에 video 태그 사용 검토
- 단일 파일 1MB 이하 유지

## 라이선스

이 디렉토리의 모든 이미지는 프로젝트와 동일하게 Apache-2.0 으로 배포됩니다.
